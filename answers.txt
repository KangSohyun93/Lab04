2. What class should implement the Comparable interface?
Lớp Media (hoặc các lớp con như Book, DVD, CD) nên implement giao diện Comparable.
3. In those classes, how should you implement the compareTo() method to reflect the ordering that we want?
Trong phương thức compareTo(), so sánh các thuộc tính như title và cost
4. Can we have two ordering rules of the item (by title then cost and by cost then title) if we use this Comparable interface approach?
Không. Giao diện Comparable chỉ cho phép một quy tắc sắp xếp duy nhất trong phương thức compareTo().

Triển khai phương thức compareTo() riêng trong lớp DVD để áp dụng quy tắc sắp xếp đặc biệt.
